Fungsi: Menghitung total biaya pengiriman berdasarkan berat, jarak, dan volume paket.
Cara Kerja:
- User memasukkan berat, tarif, dan dimensi paket (length, width, height).
- Program akan menghitung total volume, dan menentukan harga per kg sesuai jarak.
- Untuk volume > 100 cm³, Biaya tambahan Rp 50.000.
- Total biaya = (tarif × berat) + biaya volume (jika ada)
